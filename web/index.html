Satisfactory Blueprint Restoration

Recover lost blueprints from placed down instances in your save file. Everything runs locally in your browser.

Where to find files

Save files: %LOCALAPPDATA%\FactoryGame\Saved\SaveGames\<steam-id>\

Blueprints go in: %LOCALAPPDATA%\FactoryGame\Saved\SaveGames\blueprints\<session-name>\

Each blueprint needs a .sbp + .sbpcfg file pair in a subfolder matching the blueprint name.
